# Memory Game 🎮

An interactive memory card game built with vanilla JavaScript, focused on game logic, DOM manipulation, and user interaction.

Live Demo: https://ondrej108.github.io/memory-game-beta/

---

## Features

- Card matching gameplay
- 2-minute countdown timer
- Win / lose game states
- Card flip animations
- Bilingual support (EN / CZ)
- Reset and restart functionality

---

## Technologies

- HTML5
- CSS3 (animations, responsive design)
- JavaScript (DOM manipulation, game logic, events)

---

## What I focused on

- Implementing game logic in JavaScript
- Managing state (selected cards, matches, timer)
- Handling user interactions and events
- Structuring code for readability and maintainability
